# schema-dts

> TypeScript-типы для Schema.org JSON-LD.

Это русская карточка инструмента из личной SEO/marketing GitHub-базы. Репозиторий-карточка нужен, чтобы инструмент был виден как отдельный проект на GitHub, с понятным описанием и ссылками.

## Оригинальный GitHub

https://github.com/google/schema-dts

## Категория

Programmatic SEO и разработка

## Вердикт

Брать

## Что делает простыми словами

TypeScript-типы для Schema.org JSON-LD.

## Где использовать

- валидная schema
- автокомплит
- меньше ошибок Product/Article/FAQ schema

## Как поставить или подключить

```text
npm install schema-dts
```

## Идеи workflow

- Подключить к Codex/Claude/Cursor или использовать как отдельную утилиту.
- Смешивать с данными GSC, GA4, SERP, краулом сайта и таблицами.
- Превращать повторяемые SEO-задачи в workflow: аудит, мониторинг, обновление контента, pSEO, отчеты.

## Заметки

Особенно полезно в e-commerce и pSEO.
